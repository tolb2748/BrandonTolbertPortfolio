Automation of Liquid Level Control Through PID Controller Optimization

Situation: Develop a control algorithm to control the fluid level of a pump fed system for a pond.

Task: Develop a PID controller in python to control the fluid level height of a two tank gravity drainage system with an in-line pump. Use optimization techniques to tune the controller.

Action: Built process models from fundamentals and coded PID controller. Optimized controller by regression of parameters of an empirical model.

Result: Received an A+ on the project.
